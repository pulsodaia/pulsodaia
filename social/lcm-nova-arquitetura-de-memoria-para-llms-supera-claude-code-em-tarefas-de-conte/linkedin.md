Nova arquitetura de memória para LLMs supera Claude Code em tarefas de contexto longo.

Pesquisadores apresentaram o Lossless Context Management (LCM), uma abordagem determinística para gerenciar a memória em Large Language Models. O sistema demonstrou performance superior ao Claude Code em avaliações que exigem processamento de até 1 milhão de tokens.

O agente de codificação Volt, usando LCM com o modelo Opus 4.6, obteve pontuações mais altas no benchmark OOLONG para contexto longo. Esta inovação decompõe a recursão simbólica em mecanismos distintos, gerenciados pelo motor da arquitetura.

A capacidade de reter informações em contextos extensos é um desafio chave para LLMs. O LCM promete avanços significativos em eficiência e performance, abrindo novas possibilidades para aplicações de IA.
