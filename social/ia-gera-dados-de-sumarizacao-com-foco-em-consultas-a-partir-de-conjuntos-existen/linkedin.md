IA pode criar resumos sob medida, mesmo sem dados específicos.

Uma nova pesquisa no arXiv explora como gerar dados de sumarização focada em consultas a partir de datasets existentes que não as contêm. A limitação de dados para tarefas de IA mais direcionadas é um obstáculo conhecido.

O método proposto visa criar palavras-chave de consulta baseadas em evidências diretamente de conjuntos de dados de sumarização sem consulta. Isso permite que modelos de IA entendam e extraiam informações relevantes com um foco específico, tornando a sumarização mais útil.

A iniciativa busca responder se é possível gerar consultas baseadas em evidências automaticamente e se isso melhora a sumarização focada em consultas.
