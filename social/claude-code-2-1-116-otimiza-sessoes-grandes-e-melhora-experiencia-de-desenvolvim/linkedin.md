Claude Code 2.1.116 acelera o trabalho com código em até 67%.

A Anthropic lançou a versão 2.1.116 do Claude Code, focada em otimizar o manuseio de sessões de código extensas. Desenvolvedores que trabalham com projetos volumosos notarão uma melhora significativa na velocidade.

O comando `/resume`, por exemplo, agora é até 67% mais rápido para arquivos acima de 40MB. Outras atualizações visam agilizar a inicialização e a navegação em ambientes de desenvolvimento como VS Code e Cursor.

A interface também recebeu ajustes, como a integração do indicador de pensamento do Claude para um fluxo mais contínuo e a expansão da busca no comando `/config`.
