Claude Code agora guarda suas preferências de tema e editor.

A versão v2.1.119 do Claude Code introduz a persistência de configurações. Ajustes de tema, modo de editor e verbosidade agora são salvos localmente no arquivo `~/.claude/settings.json`. Isso significa que suas personalizações permanecem entre as sessões, oferecendo um fluxo de trabalho mais contínuo.

A atualização também melhora a integração com sistemas de controle de versão. A configuração `prUrlTemplate` permite direcionar links de revisão de código para plataformas customizadas, além do GitHub.com. A variável `CLAUDE_CODE_HIDE_CWD` oculta o diretório de trabalho na tela inicial.

A funcionalidade `--from-pr` foi expandida para aceitar URLs de merge requests do GitLab e pull requests do Bitbucket, além do GitHub Enterprise. O modo `--print` agora respeita as diretivas `tools:` e `disallowedTools:` do agente.
